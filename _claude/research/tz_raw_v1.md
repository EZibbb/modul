# ТЗ v1 (raw) — B2B-интернет-магазин телеком-оборудования

> Исходный текст ТЗ от Евгения, сессия 1 (2026-06-01). Хранится «как есть» для цитирования и разбора. Структурированная версия будет в `_claude/concept/*`.

**Проект:** Сайт по продаже SFP-модулей, трансиверов, патч-кордов, DAC/AOC-кабелей, CWDM/DWDM-решений.
**Целевая аудитория:** инженеры связи, сетевые администраторы, проектировщики, интеграторы, отделы закупок.
**Ключевые приоритеты:** технический поиск по параметрам, прозрачная совместимость с вендорским оборудованием, быстрый заказ, профессиональный контент.

---

## 1. Полная карта сайта

### 1.1. Главная страница
Блоки по порядку:
- **Хедер** — логотип, контакты, «Личный кабинет», корзина, поиск по артикулу/модели.
- **Экспресс-подбор совместимости** — выпадающие списки «Вендор» → «Серия/Тип устройства» → «Модель», кнопка «Показать совместимые модули» (→ каталог с фильтрацией).
- **Быстрый доступ к категориям** — карточки: SFP+, QSFP28, DAC-кабели, CWDM-мультиплексоры, патч-корды, промышленные модули.
- **Популярные решения** — подборки сценариев: ЦОД (10G/25G/100G), 5G CPRI/eCPRI, магистральные DWDM, PON.
- **Преимущества** — тестирование на реальном оборудовании, программирование под любого вендора, техподдержка инженеров.
- **Отраслевые подборки** — 3 колонки: дата-центры / операторы связи / промышленные сети.
- **Последние статьи / база знаний** — 2–3 свежих материала.
- **Футер** — меню разделов, контакты, сервис/документация, политика конфиденциальности.

### 1.2. Каталог
Верхний уровень — сетка всех категорий. Дерево — раздел 2. Каждая категория/подкатегория — страница с фильтром и товарами.

### 1.3. Инструмент «Подбор совместимости»
Отдельная страница: вендор → семейство → модель, под ним — совместимые модули с ценами и наличием.

### 1.4. Сравнение товаров
Таблица сравнения характеристик (скорость, длина волны, дальность, температура, DOM, совместимость). Удалить товар / добавить все в корзину.

### 1.5. Корзина
Список товаров, редактирование количества, удаление, промокод, «Перейти к оформлению».

### 1.6. Оформление заказа
- Юрлица: организация, ИНН, контактное лицо, оплата (счёт, карта онлайн).
- Физлица/самозанятые: упрощённая форма.
- Доставка: самовывоз, ТК, курьер.
- Загрузка реквизитов, согласие с условиями, «Подтвердить заказ».

### 1.7. Быстрый заказ по списку
Загрузка Excel (артикул, количество, желаемый вендор совместимости) → валидация → корзина / запрос КП.

### 1.8. Личный кабинет
- Профиль: компания, реквизиты, контакты.
- История заказов: фильтр по дате/статусу, повторный заказ, скачивание счёта/УПД.
- Избранное и списки: товары, списки совместимости.
- Загрузка списка для регулярных закупок.
- Настройки уведомлений: появление товара, акции.

### 1.9. Информационные разделы
- О компании — история, компетенции, сертификаты, лаборатория тестирования.
- Доставка и оплата — условия РФ/СНГ, сроки, ТК, реквизиты.
- Сервис и поддержка — программирование модулей, кастомизация кодировки, аренда тестового оборудования, входной контроль.
- Контакты — адрес офиса/склада, форма обратной связи, телефон, email.
- База знаний (блог): статьи (рубрики «Выбор модуля», «Инструкции», «Новости отрасли»), калькулятор оптического бюджета, справочник длин волн/частот (CWDM/DWDM), глоссарий (SFP28, DOM, BiDi).
- Политика конфиденциальности, Условия использования.

### 1.10. Технические страницы
404, поиск по сайту, карта сайта.

## 3. Функциональные требования

### 3.1. Поиск и фасетная фильтрация
- **Поиск:** по артикулу/названию/модели совместимости, автодополнение, поиск по характеристикам («10 км 1310 нм SFP+»).
- **Фильтры (faceted, зависят от категории):** скорость (100M…800G), тип среды/стандарт (медь RJ-45, MM SR/SR4/CSR4/eSR4, SM LR/LR4/ER/ZR/FR/PSM4, BiDi, CWDM/DWDM), дальность (до 100 м … >80 км), длина волны/канал (CWDM 1270–1610, DWDM C-band 100/50 ГГц, Ch17–Ch61), разъём (LC/SC/MPO/MTP/RJ-45/E2000), температура (0…+70 / -20…+85 / -40…+85), DOM (да/нет), совместимость с вендором (поле с автодополнением).
- Фильтры обновляют список через AJAX, рядом с чекбоксом — количество товаров.
- **Сортировка:** цена, популярность, скорость, дальность.

### 3.2. Кросс-фильтр совместимости
- **Виджет:** 3 списка «Вендор → Серия/Линейка → Модель» → URL `?vendor=cisco&model=...` → каталог с авто-фильтром. Нет совпадений → все модули форм-фактора + «Уточните совместимость».
- **Страница «Подбор совместимости»:** вендор (логотипы + поиск) → тип устройства (коммутатор/маршрутизатор/медиаконвертер/серверная карта) → модель → таблица совместимых модулей с разделением по портам (uplink 10G SFP+, downlink 1G SFP) + «Основной модуль» / «Альтернатива». Кнопка «добавить все» (комплект под модель).
- **Хранение:** связь many-to-many «товар ↔ модель оборудования» с примечанием (напр. «только с версии ПО 16.0»). Один модуль → сотни моделей.
- **Варианты заказа совместимости:** Вариант А — один SKU + опция «Прошить под…» как услуга (закладываем в ТЗ). Вариант Б — отдельные SKU на прошивку (расширение, если цена разная). В карточке: «Совместимость по умолчанию» + поле выбора прошивки «Программируется на складе».

### 3.3. Карточка товара
- Хлебные крошки + артикул.
- Галерея: фото с ракурсов, схема размеров, фото экрана тестера DOM.
- Основная инфо: название (напр. `SFP+ 10GBASE-LR 1310nm 10km LC DOM Industrial (-40…+85°C)`), цена/цена для партнёров + «Запросить оптовую цену», наличие, выбор прошивки (по умолч. «Универсальный (Generic)»), кнопки «В корзину / Купить в 1 клик / В сравнение / В избранное».
- Таблица характеристик: форм-фактор, скорость, протокол, длина волны Tx/Rx, тип лазера (DFB/FP/VCSEL), оптическая мощность min/max, чувствительность приёмника, дальность, тип волокна, разъём, DOM, температура, размеры, вес, потребление, сертификаты (CE/FCC/RoHS/MSA), артикул производителя.
- **DOM-блок:** температура модуля, напряжение, ток смещения лазера, Tx/Rx Power, аварийные флаги + скриншот консоли.
- **Таблица совместимости:** список вендоров/моделей с поиском внутри карточки, мин. версия ПО, примечания, «Проверить совместимость с вашим оборудованием».
- Документы: PDF-спецификация, сертификаты, инструкция.
- **Аналоги и похожие:** та же скорость / другая дальность; тот же форм-фактор CWDM/DWDM; аналоги других вендоров; «с этим покупают» (патч-корды, DAC, мультиплексоры).
- Отзывы (опционально).

### 3.4. Корзина и быстрый заказ
- **Корзина:** сохраняется для авторизованных, комментарий/прошивка/реквизиты на позицию, расчёт с партнёрской скидкой, прикрепить файл (ТЗ) → запрос КП.
- **Быстрый заказ:** скачать шаблон Excel (артикул, количество, вендор, примечание) → загрузить → валидация артикулов (подсветка нераспознанных, похожие) → корзина / заказ.

### 3.5. Личный кабинет
- Иерархические аккаунты компаний (админ управляет доступом сотрудников, видит их заказы).
- Списки избранного и шаблоны заказов.
- История закупок с фильтрами, повтор в 1 клик.
- Управление подпиской на поступления.

### 3.6. Сравнение товаров
Иконка «Сравнить» в списках → буфер (cookie/сессия). Страница: до 4 товаров бок-о-бок, группировка строк, подсветка различий, удалить / добавить все в корзину.

## 4. Информационные разделы

### 4.1. Сервис и поддержка
- Программирование (кодировка) модулей: процесс, список вендоров, форма заявки.
- Тестирование на оборудовании заказчика (отправить свой коммутатор на проверку).
- Кастомизация: логотип, серийные номера под проект.

### 4.2. База знаний (блог)
- **Калькулятор оптического бюджета:** сегменты кабеля, коннекторы, сварки, аттенюаторы; тип трансивера (мощность, чувствительность); вердикт «Работоспособен / Требуется усилитель»; экспорт в PDF.
- **Справочник длин волн:** CWDM (ITU-T G.694.2), DWDM (ITU-T G.694.1) — частоты, длины волн, каналы; экспорт в Excel.
- **Обучающие статьи:** «Что такое DOM», «25G SFP28 vs 10G SFP+», «Модуль для 5G CPRI», «SR4 vs LR4 для ЦОД».

## 5. Технические рекомендации

### 5.1. Платформа и технологии
- **CMS/фреймворк:** 1С-Битрикс (Бизнес/Энтерпрайз) с кастомизацией ЛИБО Laravel + Nova/Filament; для максимальной гибкости — самопис на Django (Python) или Laravel (PHP). Битрикс = готовый B2B + интеграция 1С, но много доработок для фасетного поиска и совместимости.
- **БД:** PostgreSQL (сложные связи + JSONB-атрибуты); при Битриксе — MySQL/MariaDB.
- **Поиск:** Elasticsearch / Meilisearch для мгновенного полнотекстового и фасетного поиска.
- **Фронтенд:** адаптив (Bootstrap 5 / Tailwind), динамические фильтры на Alpine.js / Vue.js.
- **Мобильная версия:** полностью адаптивна; фильтр в offcanvas-панели, крупные кнопки подбора совместимости.

### 5.2. Импорт и хранение данных
- **Импорт товаров:** из 1С:УТ/ERP в CSV/XML/CommerceML; артикул, название, атрибуты, документы, изображения.
- **Связи совместимости:** отдельная таблица «Товар – Вендор – Модель» с примечанием; пакетная загрузка CSV (артикул; vendor; model; note); связь по артикулу.
- **Вариативность прошивок:** при одном SKU — список прошивок в JSON-атрибуте, цена не зависит от прошивки.
- **Кеш/производительность:** Elasticsearch для агрегаций, кеш страниц категорий с инвалидацией по ценам/остаткам.

### 5.3. SEO-микроразметка (JSON-LD, schema.org)
- `Product` (name, sku, image, description), `Offer` (цена, InStock), `brand`.
- Совместимость через `additionalProperty` (PropertyValue «Compatible Vendors»), `isRelatedTo`.
- `BreadcrumbList`, `Article`, `FAQ`, `SoftwareApplication` (калькулятор/инструменты).

### 5.4. Безопасность и отказоустойчивость
- HTTPS, защита от DDoS (Cloudflare).
- Разграничение прав: оптовые цены скрыты от незарегистрированных B2B.
- Автоматический бэкап БД и файлов.
