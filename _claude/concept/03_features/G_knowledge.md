# G. База знаний — спецификация (11 полей)

> Витрина экспертизы + инструменты, на которые ссылаются L (ИИ) и M. Калькулятор оптбюджета — ключевой «вау».

## 1. ID + название
**G. База знаний** — G1 калькулятор оптического бюджета · G2 справочник длин волн (CWDM/DWDM) · G3 глоссарий · G4 блог/статьи.

## 2. Краткое описание
Раздел экспертного контента и интерактивных инструментов: расчёт бюджета линии, ITU-сетки, словарь терминов и обучающие статьи.

## 3. Цель
Доказать компетенцию (контент), удержать инженера на сайте и дать практические инструменты. Подпитывает ИИ-слой (RAG-контекст) и SEO.

## 4. Triggers / точки входа
- Меню «База знаний».
- Из карточки/каталога: ссылки на глоссарий (всплывающие подсказки терминов).
- Калькулятор — из M4 (конфигуратор) и L2 (ИИ-подбор) как tool.
- Главная: блок «Последние статьи».

## 5. Описание процесса
**G1 Калькулятор оптбюджета:** добавление сегментов кабеля (длина × затухание/км), коннекторов, сварок, аттенюаторов; выбор трансивера (Tx power / Rx sensitivity из каталога) → расчёт суммарного затухания, запас бюджета, вердикт «Работоспособен / Требуется усилитель». Экспорт в PDF.
**G2 Справочник длин волн:** таблицы CWDM (ITU-T G.694.2) и DWDM (ITU-T G.694.1) — частоты, длины волн, номера каналов. Экспорт в Excel.
**G3 Глоссарий:** термины (SFP28, DOM, BiDi, PSM4…) с определениями; всплывают по наведению в карточках/каталоге.
**G4 Блог/статьи:** рубрики «Выбор модуля», «Инструкции», «Новости отрасли». Статьи: «Что такое DOM», «25G SFP28 vs 10G SFP+», «Модуль для 5G CPRI», «SR4 vs LR4 для ЦОД».

## 6. UI/UX
- G1 — форма-конструктор линии (добавление компонентов), наглядная шкала запаса бюджета + вердикт, кнопка PDF.
- G2 — фильтруемые таблицы сеток, кнопка Excel.
- G3 — алфавитный список + tooltip-интеграция по всему сайту.
- G4 — список статей с рубриками, страница статьи (Article JSON-LD).

## 7. Данные (модели / сущности / API)
- **OpticalComponent**-пресеты (затухание коннектора/сварки/км по типу волокна) — seed.
- **ItuChannel**(grid[CWDM/DWDM], channelNo, freqGHz, wavelengthNm) — seed (мост к M3, фасету λ в A).
- **GlossaryTerm**(slug, term, definition) — seed; tooltip-словарь.
- **Article**(slug, title, category, body, date) — seed (3–4 статьи для демо).
- API/tools: `calc_optical_budget(segments, transceiver)` (общий с L/M), `GET /api/itu-grid`, `GET /api/glossary`.

## 8. Доступность
Полностью публично (контент + SEO).

## 9. Лимиты
- Демо: 3–4 статьи, полный глоссарий ключевых терминов, полные ITU-сетки.
- PDF-экспорт — клиентский (генерация на лету).

## 10. Этап реализации (тир)
- **P1:** G1 калькулятор оптбюджета (сильный «вау», нужен L/M) · G2 ITU-сетки · G3 глоссарий.
- **P2:** G4 блог/статьи · PDF/Excel-экспорт.

## 11. Приоритет
**P1** (G1 особенно — переиспользуется L2/M4).
