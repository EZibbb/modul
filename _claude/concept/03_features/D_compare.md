# D. Сравнение товаров — спецификация (11 полей)

## 1. ID + название
**D. Сравнение товаров** — буфер сравнения + страница сравнения (до 4 позиций).

## 2. Краткое описание
Сравнение выбранных модулей бок-о-бок по характеристикам с подсветкой различий.

## 3. Цель
Помочь выбрать между близкими модулями (LR 10км vs ER 40км, CWDM vs DWDM) без переключения между карточками.

## 4. Triggers / точки входа
- Иконка «В сравнение» в каталоге, карточке, аналогах, результатах ИИ.
- Плавающая панель «Сравнить (N)» → страница `/compare`.

## 5. Описание процесса
Добавление в буфер (cookie/сессия, для авторизованных — в ЛК). На странице — до 4 товаров колонками, строки характеристик сгруппированы (как в карточке C), **различия подсвечены**. Действия: удалить товар, добавить все в корзину, поделиться ссылкой (N3).

## 6. UI/UX
- Таблица: слева названия характеристик, справа колонки-товары (sticky-заголовки).
- Подсветка отличающихся строк; переключатель «только различия».
- Мобайл: горизонтальный скролл колонок.

## 7. Данные (модели / сущности / API)
- Буфер — клиентский (cookie/localStorage) + `GET /api/products?skus=...`.
- Группировка характеристик — общая с карточкой C (единый источник из AttributeDefinition, K).

## 8. Доступность
Публично. Для авторизованных — сохранение буфера в ЛК.

## 9. Лимиты
До 4 товаров одновременно.

## 10. Этап реализации (тир)
- **P1:** буфер + страница сравнения + подсветка различий.

## 11. Приоритет
**P1.**
