# 01 — Overview: Modul comp

> Концепция продукта. Живой документ, обновляется по ходу Этапа 1.

## Что это
**Modul comp** — B2B-интернет-магазин телекоммуникационного оборудования: оптические трансиверы (SFP/SFP+/SFP28/QSFP+/QSFP28/QSFP-DD/OSFP), DAC/AOC-кабели, патч-корды, CWDM/DWDM-оборудование, готовые решения. С техническим фасетным поиском, кросс-фильтром совместимости «вендор → модель» и **ИИ-слоем инженерного со-пилота**.

## Цель проекта (важно!)
Это **презентация-продукт** для компании, в которую Евгений планирует зайти (работа/контракт). Компания будет торговать этим оборудованием, в т.ч. перекупными позициями.

**Уровень — настоящий продукт production-качества на seed-данных** (D002), не одноразовое демо. Две оси:
- **Качество (craft)** — выкручено на максимум: чистая архитектура, реально работающие фичи и ИИ, продуктовый UX.
- **Боевитость (live)** — пока на seed/mock-данных. Реальные данные (1С, поставщики), оплата, юр-часть откроются только когда Евгений внутри компании. Продукт **спроектирован под боевой запуск** — растёт подключением реальных источников, не переписывается.

## Аудитория
Инженеры связи, сетевые администраторы, проектировщики, системные интеграторы, отделы закупок. Техническая придирчивая аудитория: ценят точность данных, плотные читаемые таблицы характеристик, быстрый параметрический поиск, отсутствие «маркетинговой воды».

## Ценностное ядро / чем отличаемся
1. **Кросс-фильтр совместимости** «вендор → серия → модель» — снимает главную боль: «какой модуль встанет в мой Cisco/Juniper».
2. **ИИ инженерный со-пилот** — тех-консультант, подбор решения по задаче, парсер конфига/ТЗ, диагностика DOM. Главный дифференциатор от FS.com и аналогов.
3. **Инженерные инструменты** — калькулятор оптбюджета, декодер названий, калькулятор экономии (совместимый vs OEM), планировщик DWDM.
4. **Инженерное доверие** — подача как datasheet: точные данные, статусы тестирования, аккуратные таблицы.

## Конкуренты и бенчмарки
**Прямые конкуренты (рынок РФ)** — кого реально обходим:
- **NAG** (nag.ru) — крупнейший, широкий каталог, но интерфейс перегружен/устарел.
- **FIBO** (Fibo Telecom) — оптика/трансиверы.
- **BAZIS** — телеком-оборудование.
- **Ланцет** — трансиверы/СКС.
Общая слабость рынка РФ: устаревший UX, слабый тех-поиск, нет нормального кросс-фильтра совместимости, нет ИИ. **Наше преимущество = современный UX + фасеты + ИИ-со-пилот.**

**Эталоны качества (к чему тянемся визуально/продуктово):**
- **FS.com** — зарубежный лидер B2B-трансиверов (каталог/карточка/совместимость) — делаем чище + добавляем ИИ, которого у них нет.
- **Linear / Vercel / Stripe** — уровень продуктовой полировки и UX.

## Стек (D003)
Next.js (App Router) + TypeScript · Tailwind + shadcn/ui · Prisma (SQLite→Postgres/Neon) · Claude API + RAG · Zod · Vercel.

## Что НЕ входит сейчас (roadmap / слайд питча)
Импорт из 1С/ERP, реальный приём оплаты, 152-ФЗ/оферта, договоры с ТК, иерархические корп-аккаунты с реальными правами, индивидуальные цены per-client, DDoS-защита. Показываем как «вот куда это растёт после подключения к вашим системам».
